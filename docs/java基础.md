## Java的8种基本数据类型

Java的基本数据类型有8种。

分别是：byte（位）、short（短整数）、int（整数）、long（长整数）、float（单精度）、double（双精度）、char（字符）和boolean（布尔值）

*字节长度*：各种数据类型占用的字节长度
